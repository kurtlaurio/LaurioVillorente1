Degree Programs

Bachelor of Science in Computer Science (BS CS)

This foundational program provides a strong understanding of core computer science principles, including algorithms, data structures, programming languages, software engineering, computer architecture, and operating systems. Graduates can pursue careers in software development, system administration, research, and more.

Bachelor of Science in Information Technology (BS IT)

This program emphasizes the practical application of technology in organizations. It covers network administration, database management, web development, cybersecurity, and system analysis. Graduates can work as network administrators, IT support specialists, web developers, and database administrators.

Bachelor of Arts in Computer Science (BA CS)

This program combines a strong foundation in computer science with courses in humanities, social sciences, and business. It prepares graduates for careers that bridge the gap between technology and other fields, such as data analysis, user experience design, and software development in specific industries like healthcare or finance.

Specialized Programs

Master of Science in Computer Science (MS CS)

This advanced program delves deeper into specific areas of computer science like artificial intelligence, machine learning, software engineering, or cybersecurity. It prepares graduates for research-oriented careers or specialized roles in the industry.

Master of Information Technology (MS IT)

This program focuses on advanced IT topics like network security, cloud computing, data analytics, and project management. It equips graduates for leadership positions in IT departments or specialized roles in specific IT areas.

Certificate Programs

Web Development

This program teaches the skills necessary to build and maintain websites and web applications, including HTML, CSS, JavaScript, and various web development frameworks.

Network Administration

This program provides the knowledge and skills to manage computer networks, ensuring their security and efficiency. Database Management: This program focuses on designing, implementing, and managing databases for storing and retrieving data effectively.

Programming Languages
